==================== 视角一：项目整体流程图 ====================

标题：脑肿瘤MRI辅助诊断系统 — 整体流程图

整个流程从左到右依次展开，分为五个大阶段：

阶段1：数据集准备（data/download.py）
- 检查本地缓存（data/raw/organized/）
  → 已存在 → 跳过下载，直接进入阶段2
  → 不存在 → 进入三级回退下载策略
    - 策略1：Kaggle API下载（kagglehub库）
      → 成功 → extract_and_organize_from_kaggle（类别名映射：glioma_tumor→glioma 等 + Training→train / Testing→test）
      → 失败 → 进入策略2
    - 策略2：GitHub镜像HTTP流式下载（requests + tqdm进度条）
      → 成功 → extract_and_organize（zip解压 + os.walk递归查找 + 目录重组为ImageFolder格式）
      → 失败 → 进入策略3
    - 策略3：NumPy合成数据生成（generate_synthetic_data）
      → 高斯噪声基底 → 椭圆形脑部轮廓模拟 → 按类别叠加不同肿瘤特征区域（胶质瘤宽亮斑 / 脑膜瘤集中亮斑 / 垂体瘤微小亮斑 / 正常无异常） → 三通道JPEG输出
- 最终输出：organized/train/{glioma,meningioma,pituitary,notumor}/ 和 organized/test/{同上} 的标准ImageFolder目录结构

阶段2：数据加载与预处理（data/dataset.py + data/preprocessing.py）
- 训练集数据变换流水线（get_train_transforms）：
  Resize(224×224) → RandomRotation(±20°) → RandomHorizontalFlip → ColorJitter(brightness=0.2, contrast=0.2) → RandomAffine(translate=0.1) → ToTensor → Normalize(ImageNet均值/标准差) → RandomErasing(p=0.25)
- 验证集/测试集数据变换流水线（get_val_transforms）：
  Resize(224×224) → ToTensor → Normalize(ImageNet均值/标准差)
- BrainTumorDataset类：遍历类别子文件夹 → 收集图像路径 → PIL验证每张图像完整性（损坏自动剔除） → __getitem__ 以RGB模式打开并应用变换
- 分层采样划分（split_dataset）：sklearn train_test_split × 2次 → 训练集70% / 验证集15% / 测试集15%，按类别标签分层保证分布一致
- DataLoader封装：训练集 shuffle=True + drop_last=True + pin_memory=True；验证/测试集 shuffle=False + pin_memory=True
- 推理专用预处理（preprocess_for_inference）：CLAHE对比度增强（LAB色彩空间L通道，clipLimit=3.0, tileGridSize=8×8） → 保持宽高比缩放(长边→224) → 中心裁剪(224×224)

阶段3：模型构建（models/transfer.py + models/cnn.py）
- 模型选择（config.py MODEL_NAME参数控制，共9种可选）：
  ├── 迁移学习模型（TransferLearningModel）：resnet50(默认) / resnet101 / resnet152 / densenet121 / densenet169 / efficientnet_b0 / efficientnet_b1 / convnext_tiny
  │    构建流程：加载ImageNet预训练权重(torchvision.models) → 可选冻结骨干网络(freeze_backbone) → 替换分类头
  │    分类头结构：Dropout(0.5) → Linear(in_features, 4)
  │    Forward路径：ResNet系列按 conv1→bn1→relu→maxpool→layer1→layer2→layer3→layer4→avgpool→flatten→classifier 逐层传播
  │    辅助方法：get_feature_vector(提取2048维特征) / unfreeze_backbone(解冻微调) / get_trainable_params(参数统计)
  └── 自定义CNN（CustomCNN）：从零训练
       结构：Initial(7×7conv stride2 + maxpool) → Stage1(Inception多尺度：1×1分支+3×3分支+5×5分支+maxpool分支 并行) → Stage2(3×3conv stride2降采样 + SE注意力模块[GAP→FC降维→ReLU→FC升维→Sigmoid→通道加权]) → Stage3(Inception + SE注意力) → GAP全局平均池化 → Classifier(Dropout0.5→Linear512→256→ReLU→BatchNorm1d→Dropout0.4→Linear256→4)
       权重初始化：Conv2d用kaiming_normal / BatchNorm用常数 / Linear用N(0,0.01)

阶段4：训练与验证（train.py）
- 训练配置（config.py）：
  ├── 损失函数：CrossEntropyLoss
  ├── 优化器：AdamW（lr=1e-4, weight_decay=1e-5）
  ├── 学习率调度：ReduceLROnPlateau（mode=max, factor=0.5, patience=5, min_lr=1e-7）
  ├── 早停机制：EarlyStopping（patience=15, min_delta=1e-4, mode=max）
  ├── 梯度裁剪：max_norm=10.0
  └── 最大训练轮数：50 epochs
- 训练循环（每个epoch）：
  ① train_one_epoch：model.train() → 遍历train_loader → inputs/labels移至GPU → forward得logits → 计算loss → backward → 梯度裁剪 → optimizer.step() → Evaluator累积统计 → ProgressBar实时显示（30字符ASCII进度条+损失+准确率+耗时）
  ② validate（@torch.no_grad）：model.eval() → 遍历val_loader → 计算val_loss和val_acc → Evaluator汇总
  ③ scheduler.step(val_acc) → 学习率衰减判定
  ④ 记录历史（7条曲线）：train_loss / train_acc / val_loss / val_acc / test_loss / test_acc / lr
  ⑤ 保存最佳模型：val_acc创新高 → torch.save(checkpoint) → checkpoints/best_model_xxx.pth（含model_state_dict + optimizer_state_dict + epoch + best_val_acc + config）
  ⑥ 早停检查：连续15 epoch无提升 → break → 恢复最佳模型参数
- 日志系统（Logger）：Python logging双通道（控制台+文件） + TensorBoard SummaryWriter（Loss/train, Loss/val, Accuracy/train, Accuracy/val, LR, Train/F1等标量）
- 训练结束后：绘制训练历史曲线图（1行3列：损失曲线+准确率曲线+学习率曲线）→ 保存JSON格式训练历史

阶段5：评估与推理
- 评估子流程（evaluate.py）：
  model.eval() + torch.no_grad() → 遍历test_loader(394张) → Evaluator累积全部预测 → compute_metrics一次性计算11项指标
  输出指标：Accuracy / Precision(Macro+Weighted) / Recall(Macro+Weighted) / F1(Macro+Weighted) / 各类别F1 / AUC-ROC(OvR) / ConfusionMatrix / ClassificationReport
  生成5张可视化图表：
    ① 混淆矩阵热力图（整数计数版 + 行归一化比例版，蓝色colormap + 单元格数值标注 + 中文类别标签）
    ② ROC曲线图（一对多策略，sklearn label_binarize + roc_curve + auc，4色曲线 + AUC标注 + 随机基线虚线）
    ③ 各类别指标分组柱状图（每类3根柱子：F1/Precision/Recall，不同颜色）
    ④ 随机样本预测网格图（16张，绿色标题=正确/红色标题=错误，显示真实类别+预测类别+置信度）
    ⑤ 训练历史曲线图（已在阶段4生成）
  所有图表保存至results/目录，DPI=150

- 推理子流程（inference.py）：
  ├── load_model_for_inference：构建模型框架 → torch.load检查点 → load_state_dict恢复权重
  ├── predict_single（单图）：preprocess_single_image(CLAHE+保持宽高比缩放+中心裁剪+标准化) → 模型forward → Softmax → topk(2)提取 → print_prediction_result（Unicode方块柱状图显示4类概率分布 + 置信度百分比）
  └── predict_batch（批量）：glob扫描目录(*.jpg/*.jpeg/*.png/*.bmp) → 逐张调用predict_single → 汇总结果列表 → 打印每张图像的文件名/预测类别/置信度
